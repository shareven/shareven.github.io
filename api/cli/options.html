v10.12.0

Underscores instead of dashes are now allowed for Node.js options as well, in addition to V8 options.

所有选项（包括 V8 选项）都允许单词用短划线（-）或下划线（_）分隔。

例如， --pending-deprecation 等同于 --pending_deprecation。

VersionChanges
v10.12.0

Underscores instead of dashes are now allowed for Node.js options as well, in addition to V8 options.

All options, including V8 options, allow words to be separated by both dashes (-) or underscores (_).

For example, --pending-deprecation is equivalent to --pending_deprecation.
