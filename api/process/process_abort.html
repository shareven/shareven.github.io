新增于: v0.7.0

process.abort() 方法会使 Node.js 进程立即结束，并生成一个核心文件。

Worker 线程中没有此特性。

Added in: v0.7.0

The process.abort() method causes the Node.js process to exit immediately and generate a core file.

This feature is not available in Worker threads.
