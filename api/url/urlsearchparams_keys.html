urlSearchParams.keys()

  • v12.14.0
  • 返回上层文档
  • 提交修改
  • 搜索
  • 返回: <Iterator>

在每一个键值对上返回一个键的 ES6 Iterator。

const params = new URLSearchParams('foo=bar&foo=baz');
for (const name of params.keys()) {
  console.log(name);
}
// 打印:
//   foo
//   foo
  • Returns: <Iterator>

Returns an ES6 Iterator over the names of each name-value pair.

const params = new URLSearchParams('foo=bar&foo=baz');
for (const name of params.keys()) {
  console.log(name);
}
// Prints:
//   foo
//   foo
